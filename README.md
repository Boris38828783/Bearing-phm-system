# 轴承故障诊断与健康管理系统（Bearing PHM System）

基于深度学习的工业级轴承故障诊断项目，涵盖数据处理、模型训练到评估的完整流程。

## 当前进展
- ✅ CWRU 数据集加载与解析
- ✅ 滑窗切分、归一化、类别平衡采样
- ✅ 1D-CNN 故障分类模型
- ✅ 跨工况评估（0hp → 3hp）
- ✅ 噪声鲁棒性测试（SNR 20 ~ -8 dB）
- 🚧 噪声增强训练

## 主要发现
- 干净数据 / 跨工况条件下准确率达 100%
- 强噪声（0 dB）下准确率降至 72.9%，且**外圈故障 81% 被误判为内圈故障**
- 原因分析：噪声破坏了外圈故障冲击的"规整性"，使其呈现出内圈故障特有的幅值调制特征

## 项目结构
