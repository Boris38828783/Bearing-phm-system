# 轴承故障诊断与健康管理系统（Bearing PHM System）

基于深度学习的工业级轴承故障诊断项目，涵盖数据处理、模型训练到评估的完整流程。

## 当前进展
- ✅ CWRU 数据集加载与解析
- ✅ 滑窗切分、归一化、类别平衡采样
- ✅ 1D-CNN 故障分类模型
- ✅ 跨工况评估（0hp → 3hp）
- ✅ 噪声鲁棒性测试（SNR 20 ~ -8 dB）
- 🚧 噪声增强训练

## 主要发现
- 干净数据 / 跨工况条件下准确率达 100%
- 强噪声（0 dB）下准确率降至 72.9%，且**外圈故障 81% 被误判为内圈故障**
- 原因分析：噪声破坏了外圈故障冲击的"规整性"，使其呈现出内圈故障特有的幅值调制特征

**噪声增强训练**将 0dB 准确率从 72.9% 提升至 100%，
  外圈故障召回率从 18.9% 提升至 100%；
  -8dB 极端噪声下仍保持 94.2%（提升 46 个百分点）

## 项目结构
src/data/ 数据加载、预处理、增强
src/models/ 模型定义
notebooks/ 探索性分析

## 环境
Python 3.10，依赖见 requirements.txt

## 部署
- 模型导出为 ONNX 格式，与 PyTorch 输出一致性误差 < 1e-6
- ONNX Runtime 推理加速 **4.84x**，单样本时延 **0.094 ms**
- FastAPI 推理服务：输入校验、置信度输出、健康检查、自动生成 OpenAPI 文档
- 提供 Dockerfile（slim 基础镜像、分层缓存优化、HEALTHCHECK 配置）
